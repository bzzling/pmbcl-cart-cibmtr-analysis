# Data Availability Statement

This project uses the publicly available CIBMTR PMBCL dataset associated with:

J. Gauthier, K. W. Ahn, J. Patel, et al., “CD19 CAR T-Cell Therapy for Primary Mediastinal Large B-Cell Lymphoma: A CIBMTR Analysis,” American Journal of Hematology 100, no. 10 (2025): 1792–1802, https://doi.org/10.1002/ajh.70033.

The raw dataset files are not included in this repository.

## Quick Start

1. Visit https://cibmtr.org/Manuscript/a02UG000004IEBtYAO/P-5954.
2. Download the dataset.
3. Extract the contents of the ZIP file and copy them into the `data/raw/` directory as is
4. Verify that there are three files, namely `datadict_pmbcl_v4.xlsx`, `pmbcl_analysis.sas7bdat`, and `Publicly Available Dataset Terms and Conditions.pdf`
